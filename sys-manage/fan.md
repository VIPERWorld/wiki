简单命令

http://blog.mixu.net/2012/01/19/how-to-thinkpad_acpi-and-fan-control-on-arch/

```
# 控制风扇
function fan() {
  sensors
  if [ $# -eq 1 ] ; then
    echo level $@ | sudo tee /proc/acpi/ibm/fan
  fi
}
```

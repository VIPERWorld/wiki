# gnome2/mate 桌面环境图标主题


* 用户自己的 ~/.icons 文件夹下
* 系统的 /usr/share/icons 文件夹下

```
.
|-- actions 操作的 增加删除,打开,取消,确认,关机,上下左右,播放暂停
|-- apps 应用程序
|-- categories 分类:办公,附件,互联网
|-- devices 设备
|-- emblems 徽征
|-- extras
|-- index.theme 文件
|-- mimetypes 文件类型:文本文件,图像,音乐
|-- places 位置,主目录,下载,回收站
|-- status 状态:声音,电量,网络信号,回收站满
`-- stock
```

# 解析html

https://github.com/wscherphof/lua-htmlparser

安装需要lua5.2

    luarocks install htmlparser

如果不知默认的版本,用下面的命令指定编译好的版本
    /usr/local/bin/luarocks  install htmlparser
    
可能需要

    apt-get install liblua5.2

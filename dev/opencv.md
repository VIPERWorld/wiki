## 编译

解压.

cmake

    cmake .

make

    make

漫长的等待(如果上天眷顾的话)编译成功

安装

    sudo make install

更行so配置 

    ldconfig

否则可能出现编译实例通过,但运行是找不到so文件,例如

```
error while loading shared libraries: libopencv_nonfree.so.2.4: cannot open shared object file: No such file or directory
```

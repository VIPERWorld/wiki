## 有用的技巧

我的所有bash脚本都以下面几句为开场白：

    #!/bin/bash
    set -o nounset
    set -o errexit

这样做会避免两种常见的问题：

* 引用未定义的变量(缺省值为“”)
* 执行失败的命令被忽略

参考 http://www.vaikan.com/bash-scripting/
